# Auth

This is a API for auth exercise.

## Installation

Add .env file to project
```bash
DB_CONNECT = 
TOKEN_SECRET = 
```

Use the package manager npm yarn or npx

```bash
npm install
```

## Usage

```bash
npm start
```

## Running

```bash
on localhost:8000
```
